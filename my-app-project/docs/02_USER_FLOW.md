# 농사일기 사용자 흐름 (USER_FLOW.md)

## 1. 사용자 흐름 개요
농사일기 앱은 사용자가 농작물 관리에 필요한 정보를 쉽게 접근하고 활용할 수 있도록 설계되었습니다. 아래는 사용자가 앱의 핵심 기능을 사용하기까지의 단계별 흐름입니다.

## 2. 사용자 흐름 단계
### 단계 1: 앱 다운로드 및 회원가입
- **행동**: 사용자는 앱 스토어에서 '농사일기' 앱을 다운로드합니다.
- **화면**: 초기 화면 → 회원가입 화면
- **설명**: 사용자 정보 입력 후 계정을 생성합니다.

### 단계 2: 농작물 정보 등록
- **행동**: 사용자는 자신의 농작물 정보를 입력합니다.
- **화면**: 농작물 등록 화면
- **설명**: 작물 종류, 재배 면적, 심은 날짜 등의 정보를 입력합니다.

### 단계 3: 홈 화면 확인
- **행동**: 사용자는 홈 화면으로 이동하여 오늘의 작업과 지역 날씨 정보를 확인합니다.
- **화면**: 홈 화면
- **설명**: 오늘 해야 할 작업 목록과 현재 날씨 정보를 한눈에 볼 수 있습니다.

### 단계 4: 작업 알림 확인
- **행동**: 사용자는 다음 작업 리스트를 확인합니다.
- **화면**: 작업 알림 화면
- **설명**: 각 작업에 대한 세부 정보를 확인하고 알림 설정을 조정합니다.

### 단계 5: 날씨 정보 조회
- **행동**: 사용자는 현재 지역의 날씨 정보와 농작물 관리 팁을 확인합니다.
- **화면**: 날씨 정보 화면
- **설명**: 날씨에 따른 관리 방법을 안내받습니다.

### 단계 6: 관리 기록 작성
- **행동**: 사용자는 작물 관리 이력을 기록합니다.
- **화면**: 관리 기록 화면
- **설명**: 관리 패턴을 분석하기 위해 이력을 입력하고 조회합니다.

### 단계 7: 병충해 진단 요청
- **행동**: 사용자는 과수의 사진을 찍어 병충해 진단을 요청합니다.
- **화면**: 과수의 병충해 진단 화면
- **설명**: AI가 진단 결과와 조치 방법을 제공합니다. 사용자는 질문 내역을 기록합니다.

### 단계 8: 피드백 및 개선
- **행동**: 사용자는 앱 사용 후 피드백을 남깁니다.
- **화면**: 피드백 화면 (선택적)
- **설명**: 사용자는 앱의 기능 개선 및 추가 요청 사항을 제출합니다.

## 3. 사용자 흐름 요약
이러한 흐름을 통해 사용자는 농사일기 앱의 핵심 기능을 손쉽게 이용하고, 자신의 농작물 관리에 필요한 정보를 적시에 확인하여 효율적인 농업 활동을 할 수 있습니다.
