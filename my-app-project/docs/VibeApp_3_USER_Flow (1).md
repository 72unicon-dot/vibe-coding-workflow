# USER_FLOW.md - 농사 도우미

## 사용자 흐름 개요
"농사 도우미" 앱은 귀농한 초보 농부가 농작물 관리 정보를 쉽고 효과적으로 활용할 수 있도록 돕기 위해 설계되었습니다. 사용자는 앱을 통해 오늘의 작업을 확인하고, 날씨 정보를 확인하며, 작물 관리 가이드를 통해 필요한 조치를 수행할 수 있습니다.

## 사용자 흐름 단계별 행동

### 1. 앱 실행
- **행동**: 사용자가 "농사 도우미" 앱을 실행한다.
- **화면**: 로그인 화면 (기존 사용자 로그인 또는 신규 사용자 가입 옵션 제공)

### 2. 홈 화면
- **행동**: 사용자가 앱에 로그인한 후 홈 화면으로 이동한다.
- **화면**: 
  - 오늘의 작업 리스트 (작업 리마인더)
  - 현재 및 예측 날씨 정보
- **사용자 행동**: 사용자는 오늘 해야 할 작업과 날씨 정보를 확인한다.

### 3. 작업 관리
- **행동**: 사용자가 작업 관리 화면으로 이동하여 작업을 추가하거나 알림을 설정한다.
- **화면**: 작업 관리 화면
  - 작업 추가 버튼
  - 기존 작업 리스트 및 알림 설정 옵션
- **사용자 행동**: 필요한 작업을 추가하고, 알림을 설정한다.

### 4. 날씨 정보 확인
- **행동**: 사용자가 날씨 화면으로 이동하여 상세한 날씨 정보를 확인한다.
- **화면**: 날씨 화면
  - 위치 기반의 실시간 날씨 정보
  - 농사에 영향을 미치는 날씨 예보
- **사용자 행동**: 사용자는 날씨 정보를 확인하고, 필요한 경우 작업을 조정한다.

### 5. 작물 관리 가이드 확인
- **행동**: 사용자가 작물 관리 화면으로 이동하여 각 과일나무에 대한 정보를 확인한다.
- **화면**: 작물 관리 화면
  - 과일나무 리스트 (사과, 배, 봉숭아, 포도 등)
  - 각 과일나무에 대한 관리 정보 및 팁
- **사용자 행동**: 사용자는 자신의 작물에 대한 정보를 확인하고, 관리 팁을 참고한다.

### 6. AI 진단 및 상담 기능 활용
- **행동**: 사용자가 상담 기능 화면으로 이동하여 작물 상태 사진을 업로드한다.
- **화면**: 상담 기능 화면
  - 사진 업로드 기능
  - AI 분석 결과 표시
- **사용자 행동**: 사용자는 사진을 업로드하고, AI의 분석 결과를 통해 필요한 관리 방법을 확인한다.

### 7. 피드백 및 개선
- **행동**: 사용자가 앱 사용 후 피드백을 제공할 수 있는 기능을 통해 의견을 제시한다.
- **화면**: 피드백 화면 (설문조사 또는 별점 평가)
- **사용자 행동**: 사용자는 앱에 대한 피드백을 남기고, 개선 사항을 제안한다.

## 흐름 요약
1. 앱 실행 ➔ 2. 홈 화면 ➔ 3. 작업 관리 ➔ 4. 날씨 정보 확인 ➔ 5. 작물 관리 가이드 확인 ➔ 6. AI 진단 및 상담 기능 활용 ➔ 7. 피드백 및 개선

이 흐름을 통해 사용자는 "농사 도우미" 앱의 핵심 기능을 쉽게 활용하고, 농작물 관리에 필요한 모든 정보를 손쉽게 접근할 수 있습니다.
