    - 텍스트: "로그인"
    - 색상: 회색

### 구성 요소
- **입력 필드**: 3개

    - 이메일 입력창

- **회원가입 버튼**: 하단 중앙에 위치, 큰 버튼

    - 색상: 테마 색상

    - 색상: 회색

    - 재배 면적 입력창
    - 심은 날짜 선택기 (달력)

    - 텍스트: "등록하기"

    - 텍스트: "취소"
    - 색상: 회색

## 4. 홈 화면
### 구성 요소
- **날씨 정보 패널**: 상단에 위치
    - 지역 이름
    - 현재 온도 및 기상 아이콘
- **작업 목록**: 중간에 위치, 카드 형식
    - 각 작업 카드: 작업 제목, 마감일, 진행 상태
- **작업 추가 버튼**: 하단 오른쪽 고정 위치, 둥근 버튼
    - 아이콘: "+" 아이콘
    - 색상: 테마 색상

---

## 5. 작업 알림 화면
### 구성 요소
- **작업 리스트**: 전체 화면을 차지하는 목록
    - 각 작업 항목: 작업 제목, 예정일, 상태
- **작업 세부 정보 버튼**: 각 항목 오른쪽에 위치
    - 아이콘: ">" 아이콘
- **알림 설정 스위치**: 각 항목 아래에 위치
    - 활성화/비활성화 스위치

---

## 6. 날씨 정보 화면
### 구성 요소
- **현재 날씨 정보**: 상단에 위치
    - 온도, 날씨 아이콘, 바람 정보
- **농작물 관리 팁 영역**: 중간에 위치
    - 텍스트 블록: 관리 팁 내용
- **새로고침 버튼**: 하단 중앙에 위치, 큰 버튼
    - 텍스트: "새로고침"
