# 🍎 과수원 일기 (Orchard Diary)

바이브코딩 AI 11단계 워크플로우를 통해 기획부터 개발, 배포까지 1일만에 완성한 초보 농부용 영농 일지 모바일 웹앱입니다.

## 🌟 주요 기능
- **직관적인 날씨 확인**: 접속 시 큼직한 글씨로 기온과 강수 확률 제공
- **작물별 추천 작업**: 사과, 배, 복숭아, 포도 탭 클릭 시 오늘의 필수 농사일 체크리스트 제공
- **영농 일지 간편 기록**: 체크박스와 간단한 메모만으로 로컬 스토리지에 자동 저장

## 🚀 사용 기술
- HTML5, CSS3, JavaScript
- Tailwind CSS (CDN)
- LocalStorage 기반 데이터 저장
- Vercel 자동 배포 연동
