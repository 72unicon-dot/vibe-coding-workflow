# 5단계. 디자인 시스템 (DESIGN SYSTEM)

- **Color Palette**: 
  - Primary (주조색): `#16a34a` (자연과 생명을 상징하는 짙은 초록색)
  - Background (배경색): `#f8fafc` (눈이 편안한 아주 밝은 회색)
  - Text (글씨색): `#1e293b` (가독성이 높은 진한 회색)
  - Accent (강조색): `#eab308` (노란색, 경고나 주의사항, 날씨 강조 시)
- **Typography**: 
  - 50대 사용자를 배려하여 모바일 화면에서도 기본 텍스트 16px, 제목 24px 이상 적용
  - 폰트 굵기를 넉넉하게 사용하여 터치와 인지가 쉽도록 설계
- **UI Elements**: 
  - 모든 버튼과 카드는 `rounded-xl` (둥근 모서리) 적용
  - 터치가 쉽도록 버튼의 높이는 최소 `48px` 이상 확보
