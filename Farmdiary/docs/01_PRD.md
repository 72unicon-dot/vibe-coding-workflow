# 2단계. PRD (제품 요구사항 정의)

## 앱 개요
과수원 일기는 초보 농부를 위한 맞춤형 영농 가이드 및 일지 앱입니다. 날씨와 작물에 맞는 오늘의 할 일을 추천하고, 완료한 일을 간단히 기록할 수 있습니다.

## 주요 기능 (MVP)
1. 날씨 대시보드: 접속 지역의 기온, 강수 확률을 크게 표시 (초보 50대 농부를 위해 텍스트 큼직하게)
2. 작물별 추천 작업: 사과, 배, 복숭아, 포도의 계절/날씨별 필수 작업(예: 적과, 방제, 수확) 체크리스트 제공
3. 영농 일지 기록: 추천 작업을 클릭하면 완료 처리되고 간단한 메모를 남겨 로컬 스토리지에 자동 저장

## 성공 지표
- 사용자가 매일 1회 이상 앱에 접속하여 영농 일지를 1개 이상 기록하는 것
