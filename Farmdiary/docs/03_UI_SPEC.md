# 4단계. 화면 명세서 (UI SPEC)

**화면: 메인 대시보드 (모바일 단일 화면 구조)**
- **헤더 영역**: 
  - 타이틀: "과수원 일기 📝"
  - 날짜: "2026년 5월 5일 (화)"
- **날씨 위젯 영역**: 
  - 아이콘: ☀️ 맑음
  - 주요 텍스트: "현재 기온 24°C / 강수 확률 0%" (글씨 크기 매우 큼, 50대 배려)
- **작물 탭 영역**: 
  - 버튼 목록: [🍎 사과], [🍐 배], [🍑 복숭아], [🍇 포도]
  - 활성화된 탭은 색상이 진하게 변경됨
- **작업 목록 영역**: 
  - 체크리스트 형태 (예: "사과나무 적과(열매솎기) 하기", "진딧물 방제약 치기")
- **메모 및 저장 영역**: 
  - 입력창: 큼직한 높이의 텍스트 영역 (placeholder: "오늘 농사 특이사항을 적어주세요")
  - 하단 버튼: 폭 100%의 두꺼운 "일지 저장하기" 버튼
