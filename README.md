# 🎯 바이브코딩 (Vibe Coding) 11단계 워크플로우

코딩을 전혀 모르는 비개발자도 AI와 소통하며 아이디어를 실제 동작하는 웹 애플리케이션으로 완성할 수 있도록 돕는 **단계별 학습 및 개발 프레임워크**입니다. 

기획(PRD)부터 디자인(UI), 개발(로직 추가), 배포까지 전체 소프트웨어 생명 주기를 11단계로 쪼개어 AI와 함께 릴레이 방식으로 개발을 진행합니다.

## ✨ 주요 기능
- **11단계 워크플로우 가이드**: 아이디어 스케치부터 시작해 명세서 작성, UI 시안 자동 생성, 실제 코드 구현 및 테스트, 피드백 반영까지 단계별로 안내합니다.
- **프롬프트 릴레이 시스템**: 현재 단계의 AI 결과물이 다음 단계의 입력(프롬프트)으로 자동 병합되어 문맥이 끊기지 않는 일관성 있는 개발이 가능합니다.
- **자동 시안 생성 및 미리보기**: AI가 생성한 HTML/Tailwind CSS 코드를 앱 내에서 즉시 렌더링하고 테스트해 볼 수 있습니다.
- **전체 파일 일괄 다운로드**: 생성된 모든 마크다운(MD) 기획 문서와 HTML 소스 코드를 단계별 파일명(`앱이름_단계_영문명.md`)으로 자동 정리하여 한 번에 다운로드할 수 있습니다.

## 🚀 시작하기

이 프로젝트는 별도의 서버 설치 없이 브라우저에서 바로 실행할 수 있는 정적(Static) 웹 애플리케이션입니다.

### 1. 로컬 환경에서 실행하기
1. 이 저장소를 `clone` 하거나 ZIP 파일로 다운로드 받습니다.
2. 폴더 안에 있는 `index.html` 파일을 크롬(Chrome)이나 엣지(Edge) 같은 웹 브라우저로 엽니다. (또는 VS Code의 Live Server 확장을 사용하면 좋습니다.)
3. 우측 상단의 **[⚙️ API 설정]** 버튼을 눌러 본인의 `OpenAI API Key`를 입력합니다. (프롬프트 실행에 필수)

### 2. 데모 모드로 체험하기
처음이신가요? 화면 우측 상단의 **[📝 데모 모드]**를 클릭해 보세요. "할일 관리 앱"의 예시 데이터가 각 단계별로 자동 채워져 전체적인 흐름을 쉽게 이해할 수 있습니다.

## 📂 폴더 구조 및 파일 설명

```text
vibe-coding-demo/
├── index.html                  # 메인 앱 화면 및 구조 (UI)
├── app.js                      # 핵심 로직 (프롬프트 릴레이, 코드 추출, 다운로드 등)
├── styles.css                  # 커스텀 스타일시트
├── data/
│   └── scenarios.js            # 11단계 워크플로우 데이터 및 예시 데모 시나리오
├── VibeCoding_User_Manual.md   # 초보자를 위한 상세 사용자 매뉴얼
└── README.md                   # 본 프로젝트 설명서
```

## 📖 사용자 매뉴얼
어떻게 시작해야 할지 막막하다면 함께 동봉된 [VibeCoding_User_Manual.md](./VibeCoding_User_Manual.md) 파일을 읽어보세요. 각 단계별 행동 요령이 쉽게 정리되어 있습니다.

## 🛠 사용된 기술 (Tech Stack)
- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **Styling**: Tailwind CSS (CDN)
- **Markdown Parsing**: Marked.js
- **Fonts**: Noto Sans KR, JetBrains Mono (Google Fonts)
